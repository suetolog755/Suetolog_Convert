# Suetolog_Convert

Бот для конвертации 3D-моделей и текстур для Black Russia (Android).

## Что умеет

### Конвертировать
- PNG → BTX (1 PNG = 1 BTX)
- Несколько PNG = несколько BTX

### Обрабатывать
- .dae → .obj
- .jbeam + .mesh → .obj
- Все PNG текстуры рядом → BTX

## Лимиты
- До 50 МБ за раз

## Канал автора
@brmodels095

## Использование
1. Подпишись на канал @brmodels095
2. Отправь файлы боту
3. Нажми «Конвертировать»
4. Получи готовые .obj и .btx файлы
