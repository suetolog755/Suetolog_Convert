# Suetolog_Convert

Бот для конвертации 3D-моделей и текстур для Black Russia (Android).

## Возможности

- 🎨 **PNG → BTX** (сжатый, 512x512, без альфа-канала)
- 📦 **.dae → .obj** (авто-упрощение до 60K вершин)
- 🧩 **.obj → .obj** (авто-упрощение)
- 🔧 **.dff → .obj** (если нужна конвертация)
- 🔧 **.3ds → .obj** (если нужна конвертация)
- 🔍 **AI-анализ** (.obj, .dae, .dff, .3ds)
- 🧠 **Советник** по совместимости с ZModeler Android

## Лимиты
- До 50 МБ за раз
