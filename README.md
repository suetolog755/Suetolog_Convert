# Suetolog_Convert

Бот для конвертации 3D-моделей и текстур для Black Russia (Android).

## Возможности

- 🎨 **PNG → BTX** (сжатый, макс 512x512, без альфа-канала)
- 🧩 **OBJ → DFF**
- 📦 **.dae / .jbeam → .obj + текстуры в ZIP**

## Лимиты
- До 50 МБ за раз

## Канал автора
@brmodels095

## Как пользоваться
1. Подпишись на канал @brmodels095
2. Отправь файл боту
3. Нажми «✅ Конвертировать»
4. Получи готовые файлы
